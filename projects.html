Projects

Here are the projects I have worked on as part of a team and for personal fun. I enjoy doing these projects as it further helps me learn more about computer science outside the classroom. These projects are sorted from latest to oldest.

Thomas - Web

Here is the first website I coded as part of the beginner web development group of De Anza's Developers' Guild. This project was made in spring 2016. The group was led by Chris Wong. The website you are on right now is this very project. It is coded in html, css, and javascript. No pictures needed, enjoy!

Rebel Invader

Revel Invader was a project created during winter 2016 as part of Developers' guild intermediate java game group. The game is coded in java using the framework Libgdx. The group was led by Vihan Chaundry with great success. The game features a space shooter type of game where you play as the rebel invading the imperial base. Shoot pellets and explode missiles to destroy enemy ships and buildings. Passive powerup activates when a limit is reached by destruction - rapid fire.

Zomblocks

This is a personal project I worked on between fall of 2015 and winter 2016. Its current state is work in progress. It uses Unity3d engine and is coded in C#. I made this after finishing some of the unity tutorials to test out their features.

Road Runner

This project was created during fall 2015 as part of Developers' guild simple games group. We programmed the game using SFML's framework and using C++ language. This is the first game I have ever coded. This group was led by Paul Merill and Vihan Chaundry. The game's theme is based on Warner Bro's "Wile E. Coyote and The Road Runner" cartoon. The game also features Sanic as a special powerup in order to go really fast!
